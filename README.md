# House Price Prediction

## Overview
This project aims to predict house prices using Machine Learning techniques. The dataset contains various house-related features such as number of bedrooms, bathrooms, area, location, and other property attributes. Data preprocessing, exploratory data analysis (EDA), outlier detection, feature engineering, and model building were performed to develop an accurate prediction model.

## Features
- Data Cleaning and Preprocessing
- Missing Value Handling
- Outlier Detection using IQR Method
- Feature Selection
- Machine Learning Model Training
- House Price Prediction

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset: House Pricing
The dataset includes:
- Number of Bedrooms
- Number of Bathrooms
- Flat Area
- Lot Area
- Number of Floors
- Overall Grade
- Basement Area
- House Age
- Renovation Year
- Zipcode
- Latitude
-  Longitude
- Sale Price (Target)
